# Dependencies
1.  Vcpkg 
2.  CMake 3.26.4 
4.  Windows 10/11 x64 
5.  Vulkan SDK
6.  Git

# Setup
-   Run these commands:  
```
git clone https://github.com/lucoiso/VulkanRender.git
cd VulkanRender
git submodule update --init --recursive
```
