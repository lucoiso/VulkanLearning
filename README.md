# Dependencies
1.  Vcpkg 
2.  CMake 3.27.4 
3.  Visual Studio 2022 Build Tools for C++ 
4.  Windows 10/11 
5.  x64 Arch 
6.  Vulkan SDK
7.  Git
8.  Python 3.11

# Setup
-   Run these commands:  
```
git clone https://github.com/lucoiso/VulkanRender.git
cd VulkanRender
git submodule update --init --recursive
```
